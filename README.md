
## Installation

```
go install github.com/devShahriar/planet@latest
```

## Usage

```bash
planet conv latlon2h3 --lat 37.775938728915946 --lon -122.41795063018799 --resolution 8
```

### Flags 

```
--lat              => latitude 

--lon              => longitude

-r or --resolution => h3 index resolution
```
